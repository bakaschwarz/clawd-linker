## Tracking plans, tasks and issues

This project uses `beads` to keep track on plans, planned tasks and work in progress.
You will use it, too.
Be sure to follow the instructions in `/.claude/rules/beads.md` when new features are planned, work is done and tasks are finished.
**ONLY** work on a task once it's tracked using `beads`!
